HTML 4

HTML 5

HTML versionen som funkar i de flesta ´browsers´, den är dock mer komplex och ser inte lika bra ut.
Senaste HTML versionen med nya tags samt med möjligheter till bättre utseende.
Längre koder som gör det svårare att kompilera enkel kod. t.ex.(<!DOCTYPE HTML PUBLIC ”-//W3C//DTD HTML 4.01/EN” ”http://www.w3.org/TR/html4/strict.dtd”>.
All kod i HTML blev simplifierad för användaren, man behöver då inte skriva lika mycket kod. (t.ex. <!DOCTYPE html> Här räcker det med att skriva denna paragraf).
Html4 har inte lika bra felhantering och kräver mer uppmärksamhet från användaren att kompilera allt på korrekt sätt.
Html5 har ett bra felhanterings system som hjälper användaren att se snabbt vad som är fel kompilerat, utöver det så kan systemet fixa detta fel automatisk.
I Html4 så finns det inte taggar som canvas, video eller audio; Men den innehåller mer taggar än Html5 som gör att kompilationen blir onödigt längre.
I Html5 har de tagit bort eller simplifierad lera taggar och addad nya som inte har funnits tidigare så som canvas, video och audio.
i Html4 så stöds multimedian av utomstående tillägg så som Silverlight eller Flash.
Html5 har inbyggt multimedia taggar i sig.
