HTML 4

HTML 5

HTML 4 är den fjärde versionen av HTML och är etablerad som standard inom webbutveckling. Den har använts i mer än tio år och är kompatibel med nästan alla webbläsare.
HTML 5 är den senaste versionen av HTML, och nuvarande standard för webbutveckling. Den har fler funktioner och attribut som hjälper till att skapa snyggare hemsidor.
Längre koder som gör det jobbigare att koda. t.ex.(<!DOCTYPE HTML PUBLIC ”-//W3C//DTD HTML 4.01/EN” ”http://www.w3.org/TR/html4/strict.dtd”>.
All kod i HTML 5 är förenklad för utvecklaren, man behöver inte skriva lika mycket kod i HTML filen. (t.ex. <!DOCTYPE html>).
I HTML 4 körs Javascript och webbsidan i samma tråd. Detta försämrar prestandan.
HTML 5 innehåller JS Web Worker Api vilket tillåter javascript och webbsidan att köra i olika trådar.
HTML 4 innehåller andra taggar än HTML 5, några exempel:
<basefont> som bestämmmer färg, storlek och font för all text i ett dokument.
<center> denna centrerar texten.
<big> gör texten lite större.
I HTML 5 har de tagit bort eller förenklat flera taggar och lagt till nya som inte har funnits tidigare, några exempel:
<time> används för att koda tid och datum.
<ruby> används tillsammans med <rt> mest för östasiatiskt språkbruk, när symboler måste skrivas ihop t.ex. 漢ㄏㄢˋ
I HTML 4 stöds multimediafunktioner av tredje part som Silverlight eller Flash. För att t.ex. Flash ska fungera behöver man installera plug-ins. Flash är dessutom inte SEO (Search Engine Optimization) vänlig.
HTML 5 har inbyggda multimedia taggar i sig. Den behöver inga plug-ins installerad och stödjs av alla webbläsare. Dessutom är dessa taggar SEO (Search Engine Optimization) vänlig.
